# Skyward: Neon Edition

A Phaser 3 neon platformer designed for GitHub Pages. The run now has three levels, timed laser gates, spike fields, patrol drones, pits, death tracking, and a full-run best time.

## Run locally

```sh
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Controls

- Arrow keys to move and jump
- Space to jump
- `R` to restart the full run

Touch controls appear on mobile devices.
